## twoplates

### Background
A two hour project to get my fingers warmed up

### Purpose
Have somewhere to put your exercise numbers to easily share

### How it works
1. Fork the repo
2. Edit your own values in `src/utilities/data.tsx`
3. Deploy (easiest with Vercel)
